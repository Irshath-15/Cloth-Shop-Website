###  Homepage
- Hero banner with promotional content  
- Featured products section  
- Navigation bar with cart count  

###  Product Listing Page
- Displays multiple clothing products  
- Basic filter and sorting UI  
- Each product links to a detail page  

###  Product Detail Page
- Large product image  
- Product description and price  
- Size selection option  
- **Add to Cart** functionality  

###  Cart Functionality
- Implemented using **localStorage**  
- Cart count visible in the navbar  
- Cart data persists after page refresh  

###  Checkout Page
- User details form (name, email, phone, address)  
- Order placement confirmation alert  
- Cart clears after order is placed  

---

##  Images
- All product images are in **JPG format**
- Stored inside the `images` folder
- Reused across pages for consistency

---

##  How to Run the Project
1. Download or clone the repository  
2. Open `index.html` in any modern web browser  
3. Browse products, add items to cart, and complete checkout
